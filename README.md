# Simple Calculator

A lightweight browser-based calculator built with vanilla HTML, CSS, and JavaScript.

## Features

- Basic arithmetic: addition, subtraction, multiplication, and division
- Decimal point support
- Clear button to reset the display
- Clean, minimal interface

## Project Structure

```
Simple_Calculator/
├── index.html   # Calculator layout and buttons
├── style.css    # Styling
└── index.js     # Calculator logic
```

## Getting Started

No build tools or dependencies required.

1. Clone the repository:
   ```bash
   git clone https://github.com/Vincentmx2000/Simple_Calculator.git
   ```

2. Open `index.html` in your browser.

That's it — the calculator is ready to use.

## Usage

| Button | Action |
|--------|--------|
| `0–9` | Enter digits |
| `.` | Decimal point |
| `+ - * /` | Arithmetic operators |
| `=` | Evaluate expression |
| `C` | Clear the display |

## Technologies

- HTML5
- CSS3
- JavaScript (vanilla)
